Závěrečná správa k projektu

Autor: Zdeněk David

Abstract: Mým úkolem bylo připravit webové stránky projektu a git hub wiki. Při plnení těchto úkolů jsem zároveň pomáhal testovat výslednou aplikaci při pořizování obrázků pro jednotlivé části webu.

Webové stránky projektu

Webové stránky jsem psal v HTML5. Nejprve jsem chtěl formátování textů vyřešit jednoduchým css souborem, ale byl jsem přesvědčen, že využít bootstrap panel a jejich přidané css bude výhodnější, efektivnější a hezčí řešení. S implementací bootstrapu mi pomohl Tomáš Madeja.

Psaní dokumentace projektu jsem často prokládal obrázky, protože je pro cílové čtenáře snazší pochopit funkcionalitu, kterou se snažím popsat, když mají možnost nahlédnutí k obrázkům aplikace.

S texty pro dokumentaci databázové části projektu mi také velice pomohl Tomáš. Ostatní části jsem zdokumentoval sám.
